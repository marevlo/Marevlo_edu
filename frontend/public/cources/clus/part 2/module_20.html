02 — Variational Deep Embedding

VaDE: A Mixture-of-Gaussians Latent Space

VaDE combines two classic ideas: variational autoencoders and Gaussian mixture models. Instead of learning a latent space first and clustering later, it forces the latent space itself to follow a mixture structure. Each cluster becomes a Gaussian cloud in the hidden space.

Why VaDE Exists

Autoencoders learn representations but ignore clustering. GMMs cluster data but struggle on high-dimensional raw inputs. VaDE solves both together by making the latent prior a Gaussian mixture.

Mixture Prior in Latent SpaceFORMULA
z ~ p(z | c)
p(z) = sum_(c=1)^K pi_c N(z | mu_c, Sigma_c)
  • K: predefined number of clusters.
  • πc: mixture weight for cluster c.
  • μc: mean of the Gaussian cluster.
  • Σc: covariance describing cluster shape and spread.
ELBO ObjectiveFORMULA
L = E_(q(z|x))[log p(x|z)] - KL(q(z|x) || p(z))
Reconstruction Term
Keeps the decoder able to rebuild the input from the latent code.
KL Term
Regularizes the encoder so its latent distribution matches the Gaussian-mixture prior.
Combined Effect
The latent space becomes both informative and cluster-shaped.
Encoder ApproximationFORMULA
q(z | x) = N(z | mu(x), sigma(x)^2)
Reparameterization Trick

The encoder outputs a mean and variance, then samples z with noise in a differentiable way so gradients can flow through the latent sampling step.

03 — VaDE Workflow and Limits

Step-by-Step Training, Code, and Failure Modes

  1. Initialize the VAE weights.
  2. Define the mixture prior in latent space.
  3. Encode mini-batches to get μ(x) and log-variance.
  4. Sample latent vectors using reparameterization.
  5. Decode and reconstruct the input.
  6. Compute the ELBO loss.
  7. Update all trainable parameters with gradient descent.
  8. Estimate posterior cluster probabilities p(c|z) for soft assignments.
Approximate Cost
O(T · n · K), dominated by neural-network forward and backward passes.
Assignment Type
Soft probabilistic cluster assignments come naturally from the generative model.
Best Use
Structured latent modeling when cluster distributions matter.
PyTorch
VAE CORE
import torch
import torch.nn as nn

class VAE(nn.Module):
    def __init__(self, input_dim, latent_dim):
        super().__init__()
        self.fc1 = nn.Linear(input_dim, 400)
        self.fc_mu = nn.Linear(400, latent_dim)
        self.fc_logvar = nn.Linear(400, latent_dim)
        self.fc2 = nn.Linear(latent_dim, 400)
        self.fc3 = nn.Linear(400, input_dim)

    def encode(self, x):
        h = torch.relu(self.fc1(x))
        return self.fc_mu(h), self.fc_logvar(h)

    def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

    def decode(self, z):
        h = torch.relu(self.fc2(z))
        return torch.sigmoid(self.fc3(h))
Limitations

VaDE is harder to train than a plain autoencoder, sensitive to the latent prior, computationally expensive, and at risk of mode collapse if some mixture components dominate the latent space.

04 — Contrastive Clustering with SCAN

SCAN: Clustering Through Neighbor Consistency

SCAN stands for Semantic Clustering by Adopting Nearest Neighbors. Instead of explicit cluster centers or latent probability distributions, it relies on self-supervised feature learning plus the rule that semantic neighbors should receive the same label.

Why SCAN Works

Contrastive pretraining makes semantic embeddings. Nearest neighbors approximate true similarity. Then a clustering head is trained so that neighbors become label-consistent.

Neighbor Consistency ObjectiveFORMULA
L = - sum_i sum_(j in N(i)) log P(y_i = y_j)
  • N(i): nearest neighbors of sample i in the current feature space.
  • yi: predicted cluster label for sample i.
  • P(yi = yj): probability that neighbors belong to the same cluster.
SCAN PipelineANIMATED
  1. Pretrain the encoder with a self-supervised contrastive method such as SimCLR or MoCo.
  2. Compute nearest neighbors in the embedding space.
  3. Attach a simple classifier head to predict provisional cluster labels.
  4. Optimize the neighbor-consistency objective.
  5. Recompute neighbors periodically and continue until assignments stabilize.
Failure Insight

If the early nearest neighbors are wrong, SCAN can reinforce those mistakes and lock onto incorrect cluster structure. Neighbor quality is therefore crucial.

05 — SCAN Code, Limits, and Comparison

Contrastive Loss, Scalability, and VaDE vs SCAN

PyTorch
CONTRASTIVE LOSS
import torch
import torch.nn.functional as F

def contrastive_loss(z_i, z_j, temperature=0.5):
    z_i = F.normalize(z_i, dim=1)
    z_j = F.normalize(z_j, dim=1)
    similarity = torch.mm(z_i, z_j.t()) / temperature
    labels = torch.arange(z_i.size(0)).long()
    return F.cross_entropy(similarity, labels)
Data Size
SCAN usually prefers larger datasets than VaDE.
Interpretability
VaDE is more interpretable because clusters are modeled as explicit Gaussian distributions.
Scalability
SCAN is usually more scalable than VaDE, though neighbor search becomes expensive at scale.
