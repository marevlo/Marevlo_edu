Autoencoder-Based Deep Clustering

Deep Embedding Clustering (DEC) moves clustering into a learned latent space. The autoencoder compresses the data, cluster centers live in that compressed representation, and KL-divergence sharpens assignments until cluster structure becomes more separable.

01 — Why Deep Clustering

Clustering Becomes Easier in a Learned Latent Space

Standard clustering methods operate directly on raw inputs, but in high-dimensional spaces distances become unreliable because of the curse of dimensionality. Points start to look almost equidistant, which makes raw-feature clustering unstable or misleading.

Core Idea

Autoencoder-based clustering first compresses the data into a latent representation where important structure is preserved and irrelevant variation is suppressed. Clustering is then performed in that learned space, not on the original raw features.

Representation Learning
An autoencoder learns a compact embedding.
Cluster Optimization
DEC refines cluster centers and assignments inside the latent space.
Advantage
Nonlinear feature extraction plus clustering objective optimization in one framework.

02 — Autoencoder and DEC Formulation

Encoder, Decoder, and Reconstruction Loss

Given a dataset X = {x₁, x₂, …, xₙ} with xᵢ ∈ ℝᵈ, the encoder maps each sample into a lower-dimensional latent vector zᵢ ∈ ℝᵏ, where k ≪ d. The decoder reconstructs the input from that latent code.

Autoencoder RepresentationFORMULA
z_i = f_theta(x_i)
xhat_i = g_phi(z_i)
L_r = sum_(i=1)^n ||x_i - xhat_i||^2
Why Reconstruction Matters

The reconstruction loss forces the encoder to retain only the information needed to rebuild the input. Noise and redundant structure are discarded, which makes the representation better suited to clustering.

Soft Assignment in Latent Space

After pretraining the autoencoder, the decoder is discarded. Cluster centers μ₁, μ₂, …, μₖ are defined in latent space, and the model computes the soft probability that latent point zᵢ belongs to cluster j.

Student’s t Soft AssignmentFORMULA
q_ij = (1 + ||z_i - mu_j||^2)^-1 / sum_(j') (1 + ||z_i - mu_j'||^2)^-1
Why Student’s t?

The Student’s t-distribution has heavier tails than a Gaussian, so distant points still exert influence. This avoids compressing everything too tightly and helps prevent the crowding problem in latent space.

03 — Target Distribution and KL Objective

Sharpening Assignments and Optimizing the Encoder

DEC sharpens the soft assignments into a target distribution that emphasizes already-confident cluster memberships. That target then becomes the optimization signal.

Target DistributionFORMULA
p_ij = (q_ij^2 / f_j) / sum_(j') (q_ij'^2 / f_j')
f_j = sum_i q_ij
KL Divergence ObjectiveFORMULA
L_c = KL(P || Q) = sum_i sum_j p_ij log(p_ij / q_ij)
Assignment SharpeningANIMATED
Important Direction

The objective is KL(P || Q), not the reverse. This matters because the sharpened target distribution becomes the anchor, forcing the network to match certainty instead of preserving ambiguity.

Key Takeaway

DEC first learns a representation, then uses its own sharpened assignments to improve that representation further. It is a self-reinforcing clustering loop inside latent space.

04 — Training Workflow

From Autoencoder Pretraining to Stable Clusters

  1. Train a standard autoencoder using reconstruction loss Lᵣ.
  2. Encode the dataset to obtain latent embeddings z.
  3. Run K-Means on the embeddings to initialize the latent cluster centers μ.
  4. Compute the soft assignments q using Student’s t-distribution.
  5. Compute the sharpened target distribution p.
  6. Compute the KL divergence loss between p and q.
  7. Use backpropagation to update the encoder and cluster centers.
  8. Repeat until assignments stabilize.

Not Suitable When

DEC is a poor fit when the dataset is small, the number of clusters is unknown, high interpretability is required, or the data are already low-dimensional and well-structured.

IDEC vs DEC

IDEC keeps reconstruction loss during clustering to prevent representation drift. That single design change addresses one of DEC’s biggest stability problems.
