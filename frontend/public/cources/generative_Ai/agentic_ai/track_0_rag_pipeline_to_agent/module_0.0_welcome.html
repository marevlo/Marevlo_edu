What You'll Build

By the time you complete the final capstone, you will have shipped one of three production builds: a research agent, a customer-support multi-agent system, or a bring-your-own-domain architecture. Each ships with eval coverage, cost telemetry, sandboxing, and HITL governance.

The agentic leap, defined

In a RAG pipeline, you decide what to do and the LLM fills in the blanks. In an agentic system, the LLM decides what to do next — picking tools, planning steps, and looping until done. The control flow itself becomes the model's output.

03 / Visual System
The ReAct Loop
Three nodes. One feedback cycle. The architectural primitive of every agent you will ever build.
Thought → Action → Observation → repeat
Diagram
Thought → Action → Observe ↺ Answer
THOUGHT
The LLM reasons about what to do next — given the goal and what it's seen so far.
ACTION
The LLM emits a structured tool call. The runtime executes it deterministically.
OBSERVE
Tool result is fed back. The LLM updates state. Loop continues — or terminates.

# Course 02 / Track 0 — conceptual scaffold.

def react_loop(goal, tools, max_steps=10):
    history = []
    for step in range(max_steps):
        thought, action = llm.decide(goal, history, tools)

        if action.name == "final_answer":
            return action.args["answer"]

        observation = tools[action.name].run(**action.args)
        history.append((thought, action, observation))

    raise RuntimeError("agent exceeded step budget")

07 / Recap
Key Takeaways
  1. A RAG pipeline has a hardcoded shape. An agent emits its own shape at run-time.
  2. The ReAct loop — Thought → Action → Observation — is the primitive every framework builds on.
  3. If you can draw the flowchart in advance, use a workflow. Agents are for the cases you can't.
  4. Agents pay 10–100× more in tokens and latency than equivalent workflows. Justify the spend.
  5. This course is layered: foundations → building → eval → production → capstone. Sequential by design.
  6. The MCP Course is a companion. Module 2.6 is the bridge. You don't need it today — you will later.
