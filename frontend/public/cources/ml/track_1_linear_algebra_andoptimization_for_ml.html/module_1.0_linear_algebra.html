Vectors, matrices &
the shapes they make

A model is a function. A function on vectors is, almost always, a linear map plus a non-linearity. Master the geometric picture — what a dot product measures, what a matrix does to space, what SVD tells you — and the algorithms in the rest of the course stop being formulas and start being shapes.

02 / Derivation
Derivation & worked example — the cosine formula and projection proof · numeric
1 · Why ⟨u,v⟩ = ‖u‖‖v‖cos θ

The algebraic definition ⟨u,v⟩ = Σ uᵢvᵢ and the geometric one look unrelated. The law of cosines bridges them. Take the triangle formed by u, v, and the side v−u opposite the angle θ:

law of cosines: ‖v−u‖² = ‖u‖² + ‖v‖² − 2‖u‖‖v‖cos θ expand the left side algebraically (bilinearity of ⟨·,·⟩): ‖v−u‖² = ⟨v−u, v−u⟩ = ‖v‖² − 2⟨u,v⟩ + ‖u‖² set the two equal, cancel ‖u‖²+‖v‖²: −2⟨u,v⟩ = −2‖u‖‖v‖cos θ ⟹ ⟨u,v⟩ = ‖u‖‖v‖ cos θ ∎

So one number carries the angle. Orthogonal ⟺ ⟨u,v⟩ = 0; that single fact is the backbone of the least-squares normal equation, PCA, and Fourier bases.

2 · The projection formula falls out of a tiny optimization

"Project v onto u" means: find the scalar α so that αu is the closest point on the line through u. Minimize the squared distance:

g(α) = ‖v − αu‖² g'(α) = −2⟨u, v − αu⟩ = 0 ⟹ ⟨u,v⟩ = α‖u‖² ⟹ α = ⟨u,v⟩ / ‖u‖² proj_u(v) = α·u key consequence: the residual (v − αu) ⊥ u ← orthogonality principle
3 · Worked example (the numbers in the demo)
u = (3, 0), v = (2, 2) ⟨u,v⟩ = 3·2 + 0·2 = 6 ‖u‖ = 3, ‖v‖ = √8 ≈ 2.83 cos θ = 6 / (3·2.83) = 0.707 ⟹ θ = 45° α = 6 / 3² = 0.667 ⟹ proj_u(v) = (2, 0) residual v − proj = (0, 2), and ⟨u, (0,2)⟩ = 0 ✓ ⊥
03 / Interactive · why L¹ produces sparsity
Norms, unit balls & the sparsity argument
A norm assigns a "length" to a vector. Different norms have different unit balls — different shapes. Constrain a loss inside one of these shapes and the shape's corners decide where the solution lands. The L¹ ball has corners on the axes; the L² ball is smooth. That's the entire geometric argument for why Lasso zeros out coefficients and Ridge doesn't.

04 / Derivation
Derivation — why L¹ zeros coefficients but L² only shrinks them subgradient proof

The geometric "corner" picture is the intuition; the subgradient optimality condition is the proof. Write the penalized objective and ask: when is the optimum exactly at wⱼ = 0?

J(w) = R̂(w) + λ Ω(w) stationarity: 0 ∈ ∂J(w)
L¹ case: Ω(w) = ‖w‖₁

The function |wⱼ| is not differentiable at 0; its subdifferential there is the whole interval [−1, 1]. Stationarity in coordinate j reads:

0 = ∂R̂/∂wⱼ + λ·s, s ∈ [−1, 1] (when wⱼ = 0) ⟹ wⱼ = 0 is optimal ⟺ | ∂R̂/∂wⱼ | ≤ λ

So a whole range of gradients — anything with magnitude below λ — pins the coefficient to exactly zero. That is sparsity: raise λ and more coordinates fall inside the band. (This same condition is the soft-threshold rule you'll implement for Lasso coordinate descent in Track 3.)

L² case: Ω(w) = ½‖w‖₂²
∂Ω/∂wⱼ = wⱼ (smooth, single-valued) 0 = ∂R̂/∂wⱼ + λ wⱼ ⟹ wⱼ = −(1/λ)·∂R̂/∂wⱼ

Now wⱼ = 0 requires ∂R̂/∂wⱼ = 0 exactly — a measure-zero coincidence. Ridge therefore shrinks every coefficient toward zero but essentially never sets one to it. The kink at the origin is the entire difference.

07 / Derivation
Derivation — where the SVD comes from, with a worked 2×2 and NumPy proof · numeric · code
1 · Build it from the eigendecomposition of AᵀA

For any real A (m×n), the matrix AᵀA is symmetric and positive semidefinite, so the spectral theorem gives orthonormal eigenvectors v₁…vₙ with eigenvalues λᵢ ≥ 0. Define the singular values and left vectors:

σᵢ = √λᵢ uᵢ = A vᵢ / σᵢ (for σᵢ > 0)

The uᵢ are automatically orthonormal — that is the crux:

⟨uᵢ, uⱼ⟩ = (vᵢᵀ AᵀA vⱼ)/(σᵢσⱼ) = (λⱼ vᵢᵀvⱼ)/(σᵢσⱼ) = (λⱼ/σᵢσⱼ)·δᵢⱼ = δᵢⱼ

Since A vᵢ = σᵢ uᵢ for every i, stacking the columns gives A V = U Σ, and because V is orthogonal (VᵀV = I):

A = U Σ Vᵀ — exists for every matrix (unlike eigendecomposition)
2 · Worked example
A = [ 2 1 ] AᵀA = [ 5 4 ] [ 1 2 ] [ 4 5 ] eigenvalues of AᵀA: λ = 9, 1 ⟹ σ₁ = 3, σ₂ = 1 v₁ = (1, 1)/√2, v₂ = (1, −1)/√2 u₁ = A v₁/σ₁ = (3,3)/√2 / 3 = (1,1)/√2 u₂ = A v₂/σ₂ = (1,−1)/√2 / 1 = (1,−1)/√2 check: σ₁σ₂ = 3 = |det A| ✓ σ₁/σ₂ = 3 = condition number κ(A)
3 · In NumPy — and rebuilt by hand
import numpy as np A = np.array([[2., 1.], [1., 2.]]) # library SVD U, s, Vt = np.linalg.svd(A) # s -> [3., 1.] ; A ≈ U @ np.diag(s) @ Vt # rebuild from the eigendecomposition of AᵀA (the proof, in code) lam, V = np.linalg.eigh(A.T @ A) # ascending order order = np.argsort(lam)[::-1] lam, V = lam[order], V[:, order] sigma = np.sqrt(np.clip(lam, 0, None)) Uman = (A @ V) / sigma # columns uᵢ = A vᵢ / σᵢ assert np.allclose(Uman @ np.diag(sigma) @ V.T, A) # ✓ reconstruction # condition number = σ_max / σ_min (how much A amplifies error) kappa = sigma[0] / sigma[-1] # -> 3.0

Why ML cares: truncating to the top-k singular values is the best rank-k approximation (Eckart–Young) — that is PCA, latent-semantic analysis, and image compression. And κ(A) = σ_max/σ_min predicts how badly a near-singular XᵀX will blow up the OLS solution — the motivation for ridge.

08 / Two more shapes you'll meet everywhere
PSD matrices & the RKHS preview
Two ideas worth bookmarking now — both reappear constantly in the algorithms ahead.

Positive (semi-)definite matrices

A symmetric matrix M is positive definite if xᵀMx > 0 for every non-zero x. Geometrically: xᵀMx draws a bowl-shaped quadratic — never below zero, with a unique minimum at the origin.

You'll meet them as covariance matrices (Σ), kernels (K), and Hessians at minima — every time the quadratic form has to point upward.

appears in: Gaussian Naive Bayes · GP · Kernel SVM · Newton's method

Reproducing Kernel Hilbert Space

An RKHS is an inner-product space of functions where the inner product is computed by a kernel k(x, x') — never by explicitly building the feature map.

This "kernel trick" is what lets kernel SVMs draw non-linear boundaries and Gaussian Processes reason over functions instead of points.

09 / Derivation
Practice — prove it, compute it exercises + solutions
  1. Compute the eigenvalues of M = [[2,1],[1,2]] by hand.

    Characteristic polynomial: det(M−λI) = (2−λ)² − 1 = 0 ⟹ (2−λ) = ±1 ⟹ λ = 1, 3. Eigenvectors: (1,−1) for λ=1, (1,1) for λ=3.

  2. Show M = [[2,1],[1,2]] is positive definite directly from the definition xᵀMx > 0.

    xᵀMx = 2x₁² + 2x₁x₂ + 2x₂² = x₁² + x₂² + (x₁+x₂)². A sum of squares, and it is zero only when x₁ = x₂ = 0. Hence PD. (Equivalently: both eigenvalues 1, 3 are > 0.)

  3. For any data matrix X, prove XᵀX is positive semidefinite. Why does this guarantee the OLS objective is convex?

    For any z: zᵀ(XᵀX)z = (Xz)ᵀ(Xz) = ‖Xz‖² ≥ 0. So XᵀX ⪰ 0. The OLS Hessian is 2XᵀX ⪰ 0, and a function with PSD Hessian everywhere is convex — so any stationary point is a global minimum. (Strictly positive definite ⟺ X has full column rank ⟺ a unique solution.)

  4. Let r = y − Xŵ be the OLS residual. Show Xᵀr = 0 and state what it means geometrically.

    The normal equations are XᵀXŵ = Xᵀy ⟹ Xᵀ(y − Xŵ) = Xᵀr = 0. Geometrically the residual is orthogonal to every column of X — i.e. ŷ = Xŵ is the orthogonal projection of y onto the column space of X. This is the dot-product orthogonality principle at full scale.
