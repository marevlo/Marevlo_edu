02 / CORE
Core Concepts

What Is a Random Experiment?

A random experiment is any process whose outcome cannot be predicted with certainty, even if you repeat it under identical conditions. The key feature: you know what could happen, but not what will happen.

Three things must be true for a process to count as a random experiment:

① The experiment can be repeated (at least in principle).
② All possible outcomes are known in advance.
③ The specific outcome of a single trial is uncertain.

Examples: Flipping a coin, rolling a die, drawing a card, measuring a stock price tomorrow, an MRI machine reading a noisy signal, a neural network producing a prediction with uncertainty — all are random experiments.

Sample Space (Ω)

The sample space, written Ω (omega) or S, is the set of every possible outcome of a random experiment. It is the "universe" inside which all probability lives.

Coin toss → Ω = {H, T}
Die roll → Ω = {1, 2, 3, 4, 5, 6}
Two coin tosses → Ω = {HH, HT, TH, TT}
A continuous quantity (temperature) → Ω = an interval of real numbers

Why it matters: If your sample space is wrong, every probability you compute will be wrong. In machine learning, defining Ω correctly is equivalent to defining what the model is allowed to output.

Events

An event is any subset of the sample space — a collection of outcomes we care about. We say "the event occurred" if the actual outcome of the experiment lies inside that subset.

Rolling a die, Ω = {1,2,3,4,5,6}:

• Event A = "roll an even number" = {2, 4, 6}
• Event B = "roll greater than 4" = {5, 6}
• Event C = "roll a 7" = { }  (impossible event)
• Event D = "roll any number" = Ω  (certain event)

Events can be combined using set operations: A ∪ B (A or B), A ∩ B (A and B), Aᶜ (not A).

Probability as a Function

Probability is a function P that assigns to every event a real number between 0 and 1, capturing "how likely" that event is. P : Events → [0, 1].

P(A) = 0 means A is impossible. P(A) = 1 means A is certain. Everything else lives on the spectrum of uncertainty in between.

Mental model: Think of Ω as a 1-unit pie. Probability distributes that pie across all events. The total slice can never exceed 1, and we never have negative slices.

04 / FORMAL
Mathematical Foundation
⟁ Classical Probability (Equally Likely Outcomes)
P(A)  =  |A| / |Ω|

|A| is the number of outcomes favoring event A. |Ω| is the total number of outcomes in the sample space. This is the original definition Laplace used in the 1800s — and it still drives countless combinatorial calculations today.

⟁ The Three Axioms of Probability (Kolmogorov, 1933)
  1. Non-negativity: For any event A,  P(A) ≥ 0. Probability is never negative — uncertainty has no "negative weight".
  2. Normalization:  P(Ω) = 1. Something in the sample space must happen. The total probability of all outcomes is exactly 1.
  3. Countable additivity: If A₁, A₂, A₃, … are mutually exclusive (no overlap), then P(A₁ ∪ A₂ ∪ …) = P(A₁) + P(A₂) + …

Every probability you have ever seen — from coin flips to deep learning loss functions — is a direct consequence of these three rules. They are the constitution of probabilistic reasoning.

⟁ Consequences of the Axioms
P(∅) = 0   |   P(Aᶜ) = 1 − P(A)   |   0 ≤ P(A) ≤ 1
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

The inclusion–exclusion principle: when events overlap, we subtract the intersection to avoid counting it twice. Try it mentally on a Venn diagram — the math becomes a picture.

05 / WORLD
Real-World Applications

Machine Learning ML

Every classifier outputs P(class | features). The softmax layer in a neural network is literally a probability distribution over the sample space of class labels.

Reinforcement Learning RL

Agents pick actions using probability distributions called policies — π(a|s) — turning Ω into a space of possible behaviors.

Bayesian AI AI

Bayesian networks model entire systems as joint probability distributions over events, enabling reasoning under uncertainty.

Finance & Risk FIN

Stock returns, option pricing, and Value-at-Risk calculations all sit on sample spaces of possible future market states.

Cybersecurity SEC

Spam filters, anomaly detection, and intrusion detection treat suspicious events as probabilistic deviations from normal traffic.

Medical Diagnostics MED

Test sensitivity and specificity are pure probability statements — events defined on the sample space of patient states.

Quantum Systems QM

In quantum mechanics, the squared amplitude |ψ|² is a probability density — Ω becomes the space of measurable physical states.

Recommendation Engines REC

Netflix, Spotify, Amazon predict P(user likes item) — defining events on a vast sample space of user-item interactions.
