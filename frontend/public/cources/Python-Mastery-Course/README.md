# Python Mastery — Complete Course

A complete, beginner-to-expert Python course delivered as runnable Jupyter notebooks.
Every notebook follows the same structure: learning objectives, prerequisites, five
explanatory parts with commented runnable code, six graded worked examples (two easy,
two medium, two difficult), four to five exercises with worked solutions, and a summary.
Every notebook has been executed end to end.

58 notebooks across 9 tracks.

## How to use

Open any notebook in Jupyter (`jupyter notebook` or `jupyter lab`) or VS Code.
Run cells with Shift+Enter. Tracks are designed to be followed in order, though
each notebook also stands alone. Python 3.12 is assumed.
