Part 01

Item-Item Cosine Similarity

From M09 — sim(i,j) = cos(r_i, r_j) where r_i is item i's column in R.

Two items are similar if the same users rated them similarly. The cosine similarity between item vectors captures this: items rated by the same group of users will have vectors pointing in the same direction.

The key trick: RTR gives us all pairwise dot products in one matrix multiply — the (i,j) entry is the dot product of item i and item j's rating vectors, which is also the number of users who rated both items.

Lab_03 · item cosine similarity
def compute_item_similarity(R, shrinkage=0, k_neighbors=50):
    n_items    = R.shape[1]
    item_norms = np.sqrt(np.array(R.power(2).sum(axis=0)).flatten())
    item_norms[item_norms == 0] = 1.0

    # R^T @ R gives (i,j) = dot product of item i and item j vectors
    # = number of users who rated BOTH items (for binary matrices)
    RtR       = (R.T @ R).toarray()
    co_counts = RtR.copy()    # co-interaction counts

    # Cosine: divide by product of norms
    norms_outer = np.outer(item_norms, item_norms)
    cosine_sim  = RtR / norms_outer
    np.fill_diagonal(cosine_sim, 0.0)   # item is not its own neighbor

    return cosine_sim
Part 02

Shrinkage Regularization

From M11 — the bug: items with 1 co-rater get sim=1.0. The fix: shrink by co-count.

Without shrinkage, two items rated by only a single user in common get a similarity of 1.0 — the maximum possible. But one co-rater is not enough evidence to trust a similarity estimate. Shrinkage penalizes low co-count similarities by multiplying by nij/(nij + λ).

📐 The Shrinkage Formula

sim_shrunk(i, j) = [n_ij / (n_ij + λ)] × cos(i, j)

When n_ij → ∞, the factor → 1 (no penalty, fully trust the similarity).
When n_ij = 0, the factor = 0 (no co-raters, zero similarity).
λ controls how quickly we "trust" co-ratings — typical values: 50–200.

Lab_03 · shrinkage applied
    # Apply shrinkage weight: n_ij / (n_ij + λ)
    shrink_w   = co_counts / (co_counts + shrinkage)  # element-wise
    similarity = cosine_sim * shrink_w                # shrunk similarity

    # Sparsify: keep only top-k neighbors per item (memory efficiency)
    for i in range(n_items):
        row = similarity[i]
        if np.count_nonzero(row) > k_neighbors:
            threshold    = np.partition(row, -k_neighbors)[-k_neighbors]
            row[row < threshold] = 0.0

    return similarity

def item_knn_recommend(user_idx, R_train, similarity, k=10):
    interacted = np.where(R_train[user_idx].toarray().flatten() > 0)[0]
    if len(interacted) == 0: return []
    # Score = sum of similarities to all interacted items
    scores             = similarity[interacted].sum(axis=0)
    scores[interacted] = -np.inf    # exclude already-seen items
    return np.argsort(scores)[-k:][::-1]
